# Header Presets

### What a Header Preset Is

A Header preset stores a set of predefined design and configuration settings for your header. Instead of configuring the header repeatedly across layouts or pages, you can save one preset and reuse it wherever needed.

### Key Elements in the Preset

When creating a preset, you define:

* **Title**
  The name of the preset (e.g., “Main Header”, “Transparent Header”, etc.)

* **Image**
  A visual thumbnail to help identify the preset quickly in the list

* **Description**
  Optional notes explaining the purpose or style of this header

### How It’s Used for Headers

Once created, a Header preset can include settings such as:

* Background color or image
* Sticky behavior
* Logo position
* Menu alignment
* Padding and spacing
* Responsive behavior

You can then apply this preset to your header layout, allowing consistent styling across multiple pages.

### Why Use Header Presets

* **Consistency**: Keep the same header style across your site
* **Efficiency**: Avoid reconfiguring settings each time
* **Flexibility**: Easily switch between different header styles (e.g., dark vs light header)

### Typical Workflow

1. Create a preset (like in your screenshot)
2. Configure header layout settings
3. Save the preset
4. Apply it to any layout or page where needed
