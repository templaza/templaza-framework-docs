# Technical Requirement
## Requirements:
- Joomla: 4.x, 5.x
- PHP: 8.2+
- Mysql: 8.0+ or Maria DB 10.4+

## PHP Specifications:
- max_execution_time = 300
- max_input_time = 300
- max_input_vars = 2000
- memory_limit = 512M
- post_max_size = 10M
- upload_max_filesize = 10M

## Browser Support
- Chrome 64+
- Firefox 58+
- Edge 14+
- Safari 10+
